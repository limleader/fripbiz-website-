# 프립비즈 웹사이트 리디자인 설계 문서

**날짜**: 2026-03-02
**참고 사이트**: https://fripbiz.oopy.io/
**목적**: 기업 고객 대상 B2B 프로그램 소개 및 단체 문의 접수 웹사이트

---

## 개요

기존 프립비즈(Frip Biz) 사이트의 콘텐츠를 그대로 유지하되, 더 생동감 있고 현대적인 UI로 재구성. 기업 HR 담당자가 서비스를 둘러보고 문의를 보낼 수 있는 랜딩 페이지.

---

## 기술 스택

| 항목 | 선택 |
|------|------|
| 프레임워크 | Next.js 15 (App Router) |
| 스타일링 | Tailwind CSS |
| 애니메이션 | Framer Motion |
| 이메일 발송 | Resend API |
| 배포 | Vercel |
| 언어 | TypeScript |

---

## 페이지 구조

단일 페이지(SPA) 구성:

```
헤더 (고정 네비게이션)
  ├── 로고 (프립비즈)
  ├── 메뉴: 서비스 소개 | 프로그램 | 프로세스 | 문의
  └── CTA 버튼: "견적 문의하기"

1. 히어로 섹션
   ├── 메인 카피: "조직에도 인사이트가 필요합니다"
   ├── 서브카피: 조직 효율성·생산성 향상 메시지
   ├── CTA 버튼: "프로그램 둘러보기" / "견적 문의하기"
   └── 배경: 그라디언트 + 플로팅 도형 애니메이션

2. 서비스 3종 카드 섹션
   ├── 인사이트 렉처 (Insight Lecture) - 직무 역량 교육
   ├── 인사이트 워크숍 (Insight Workshop) - 5가지 솔루션
   └── 인사이트 트립 (Insight Trip) - 맞춤형 워크숍 여행

3. 프로세스 섹션 (5단계 타임라인)
   문의 접수 → 솔루션 설계(1-2회 미팅) → 맞춤형 제안 → 프로그램 진행+만족도조사 → 최상의 기업복지

4. 문의 폼 섹션
   ├── 회사명, 담당자 이름, 이메일, 전화번호
   ├── 관심 서비스 (체크박스: 렉처/워크숍/트립)
   ├── 문의 내용 텍스트에어리어
   └── 제출 버튼 + 성공/실패 메시지

푸터
   ├── 로고, 연락처 (02-6218-0031 / b2b@frientrip.com)
   └── 저작권
```

## UI/UX 디자인 방향

### 스타일
**생동감 있고 밝은 "툴트론" 스타일**

### 컬러 팔레트
| 용도 | 색상 | 헥스 |
|------|------|------|
| Primary | 에너지 오렌지 | `#FF6B35` |
| Secondary | 보라 | `#7C3AED` |
| Accent | 민트 그린 | `#06D6A0` |
| 배경 | 밝은 화이트 | `#FAFAFA` |
| 텍스트 | 딥 네이비 | `#1A1A2E` |

### 타이포그래피
- 헤딩: `Pretendard` (한국어 모던 폰트)
- 본문: `Noto Sans KR`

### 애니메이션 (Framer Motion)
- 스크롤 시 섹션 페이드-인 (IntersectionObserver)
- 서비스 카드 호버 시 위로 올라오는 효과 (y: -8px)
- 히어로 배경 플로팅 도형 (부유 애니메이션)
- 프로세스 타임라인 순차 등장

---

## 문의 폼 연동

- Next.js API Route (`/api/contact`) 서버사이드 처리
- Resend API로 이메일 발송 (무료: 월 3,000건)
- 입력 유효성 검사 (클라이언트 + 서버)
- 제출 후 성공/실패 피드백 UI

---

## 배포 계획

1. GitHub 레포지토리 생성
2. Vercel 프로젝트 연결
3. 환경변수 설정: `RESEND_API_KEY`
4. 메인 브랜치 push → 자동 배포
